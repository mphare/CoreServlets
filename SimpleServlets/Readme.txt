Examples of Simple Servlets. No raw HTML files.

http://localhost:8080/SimpleServlets/hello - HelloWorld - No HTML generation : http://localhost:8080/SimpleServlets/hello
http://localhost:8080/SimpleServlets/testServlet - TestServlet - Simple message and header in HTML
http://localhost:8080/SimpleServlets/testServletCSS - TestServletCSS - Same as TestServlet but uses CSS (and includes overlib.js)
http://localhost:8080/SimpleServlets/testServletUtils - TestServletUtils - Same as TestServletCSS but uses a Utility Java Class to generate some of the HTML Headers
